# Stock Performance Analysis: Apple vs Tesla (R Project)

## Project Overview
This project analyzes and compares the financial performance, risk, and return characteristics of Apple (AAPL) and Tesla (TSLA) stocks using R.  
The analysis focuses on return calculations, volatility measurement, risk–return trade-offs, correlation, and visual exploration of stock behavior over the last 5 years.

This project demonstrates practical skills in financial data analysis, statistical computation, and data visualization using R.

---

## Objectives
- Compare long-term performance (CAGR) of Apple and Tesla
- Analyze daily and monthly stock returns
- Measure stock risk using volatility
- Evaluate risk–return trade-offs using Sharpe Ratio
- Understand investment behavior of growth vs stable stocks
- Visualize and interpret financial data insights
